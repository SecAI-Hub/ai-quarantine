# Contributing

Contributions are welcome. Please follow these guidelines:

## Getting started

```bash
git clone https://github.com/SecAI-Hub/ai-quarantine.git
cd ai-quarantine
pip install -e ".[dev]"
python -m pytest tests/ -v
```

## Guidelines

- All changes must include tests
- Run `python -m pytest tests/ -v` before submitting
- Follow existing code style (no auto-formatters that change unrelated lines)
- Security-sensitive changes require review from a maintainer

## Pull requests

1. Fork the repository
2. Create a feature branch
3. Write tests for new functionality
4. Submit a PR with a clear description

## Code of conduct

Be respectful. Focus on the technical merits of contributions.
